GitHub Repo to Text Prompt

This tool generates a text prompt from a GitHub repository using URL parameters.

Usage: Append ?url= with a GitHub repository URL and optionally &token= for private repos.

Examples:

  • Public repo: /generate?url=https://github.com/octocat/Hello-World
  • Specific branch: /generate?url=https://github.com/octocat/Hello-World/tree/master
  • Private repo: /generate?url=https://github.com/owner/private-repo/tree/dev&token=your_token

All file contents are displayed as raw text and can be copied directly.
